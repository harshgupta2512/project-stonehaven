# DESIGN SYSTEM
## Premium Residential Brochure

---

> *A complete design reference for the creation of a world-class luxury residential property brochure. Every decision documented here is intentional. Restraint is the strategy.*

---

## 01 — BRAND PHILOSOPHY

This brochure exists at the intersection of architecture, nature and refined living. Every design decision — from the weight of a typeface to the thickness of a rule — must reinforce a single idea:

**This is not a home. This is a life, considered.**

The design language draws from the materiality of the homes themselves. Stone. Timber. Olive. Slate. The palette is tactile. The typography is assured. The layouts breathe.

## 03 — TYPOGRAPHY

### Typeface Selection

| Font | Role | Rationale |
|---|---|---|
| **Clash Display** | Display, Headlines, Quotes | Geometric editorial character. Premium without coldness. Commands attention. |
| **Visby CF** | Body, Captions, Labels, UI | Clean, warm, highly legible. Supports without competing. |

---

### Weight System

#### Clash Display Weights in Use

| Weight | Usage |
|---|---|
| Extralight | Cover project name, large graphic background text |
| Light | Subheadings at rest, secondary display moments |
| Regular | Pull quotes, mixed-case display, introductory paragraphs |
| Medium | Section opener titles, chapter headers |
| Semibold | Room names, amenity titles, floor plan labels |

> **Never use Bold or Extrabold.** At luxury price points, excessive weight reads as noise, not confidence.

---

#### Visby CF Weights in Use

| Weight | Usage |
|---|---|
| Light | Captions, image credits, disclaimer, distances |
| Regular | All body copy, venue descriptions, collaborator text |
| Medium | Small labels, contact details, key callouts |

---

### Type Scale

```
Cover project name      Clash Display Extralight    72 – 96pt
Section opener title    Clash Display Medium        48 – 64pt
Pull quote              Clash Display Regular       36 – 48pt
Page subheading         Clash Display Semibold      16 – 20pt
Room / venue title      Clash Display Semibold      14 – 16pt
Body copy               Visby CF Regular             9 – 11pt
Captions                Visby CF Light               7 – 8pt
Small labels / tags     Visby CF Medium              7pt
Disclaimer              Visby CF Light               6 – 7pt
```

---

### Typography Rules

```
Alignment        →  Always left-aligned. Never centred body text. Never justified.
Line height      →  1.6 – 1.8× the point size for body copy
Column width     →  Maximum 55 – 60 characters per line
Letter spacing   →  Section labels: +150 tracking. Headlines: +30 – 80 tracking.
Body tracking    →  0 to +10. Never negative track body copy.
Paragraph spacing→  Space after paragraphs only. No indents.
Hyphenation      →  Off. Always.
```

---

### Label / Tag Format

All small section labels and category tags follow this format:

```
VISBY CF MEDIUM — 7PT — ALL CAPS — +150 TRACKING — SAGE OLIVE #696B55

Example:
PART ONE  ·  LOCATION
01  ·  ARCHITECTURE
```

---

### Pull Quote Format

```
Clash Display Regular
36 – 48pt
Mixed case (not all caps)
Warm Stone or Sage Olive tone
Generous leading — 1.3×
Attribution below in Visby CF Light 8pt, tracked +100

Example:
"A place where the everyday
becomes effortless."

— Architect Name, Studio Name
```

---

### Graphic Type Treatment

On section opener pages, use Clash Display Extralight at 180 – 240pt as a purely graphic background element at 6 – 10% opacity. This creates depth and texture without adding visual noise.

```
Example: "LOCATION" set at 200pt, #E8E8E8, 8% opacity
         sitting behind the section image on an olive ground
```

## 04 — LAYOUT SYSTEM

### Page Dimensions

```
Format          →  A4 Portrait  (210 × 297mm)  or  Custom 230 × 310mm
Bleed           →  3mm all sides
Margins         →  Outer: 20mm  Inner: 25mm  Top: 22mm  Bottom: 28mm
Columns         →  2-column grid with 6mm gutter
Baseline grid   →  12pt
```

---

### Grid Principles

- The grid is a guide, not a cage. Premium layouts occasionally break it — intentionally.
- Full-bleed images are always truly full-bleed. No partial bleeds.
- Text never sits closer than 12mm to a full-bleed image edge.
- Asymmetric layouts: image takes 60% of spread, copy takes 40%.
- Allow generous white space. Negative space is a design element, not wasted paper.

---

### Spread Types

#### Type A — Hero Spread
```
Full bleed image across both pages
Single short text block bottom right or left
Caption bottom corner
Used for: Section openers, standout architecture renders
```

#### Type B — Split Spread
```
Left page:  Full bleed image (portrait or square crop)
Right page: White/stone background + headline + body copy
Used for: Room descriptions, suburb story, architect quotes
```

#### Type C — Editorial Grid
```
Multiple images in a composed grid (not uniform)
Short copy block anchoring one corner
Used for: Material palette, dining highlights, lifestyle
```

#### Type D — Copy Dominant
```
Minimal or no image
Rich typographic hierarchy
Thin rule accents, section labels
Used for: Sustainability initiatives, floor plan supporting text
```

#### Type E — Floor Plan
```
Clean white background
Plan drawing centred with generous margin
Home type label top left
Room labels, areas, north point
No decorative elements — let the plan speak
```

---

### Spacing Scale

```
XS   →   4mm    Inline spacing, between icon and label
S    →   8mm    Between caption and image
M    →  12mm    Between paragraphs / sections within a page
L    →  20mm    Between major content blocks
XL   →  32mm    Top of page to first element (after margin)
XXL  →  48mm    Section to section breathing room
```

## 05 — RULES & DIVIDERS

Rules do enormous quiet work in premium layouts. Use them consistently.

```
Weight          →  0.25pt hairline. Never thicker than 0.5pt.
Length          →  Full column width or custom short (20 – 40mm accent rule)
Colour by use:

  B9AFA3  →  General content dividers, amenity listing separators
  696B55  →  Section introductions, emphasis moments, pull quote above/below
  553B29  →  Around floor plans, material detail borders
  E8E8E8  →  On dark pages — separating reversed-out content
```

## 06 — IMAGE DIRECTION

### Photography Style

- **Tone:** Warm, golden, natural light. Dusk and morning preferred over midday.
- **Mood:** Unhurried. Spacious. Never lifestyle-catalogue obvious.
- **People:** Minimal or absent. Suggest life without showing it.
- **Colour:** Images should feel consistent with the palette — olive greens, warm stones, deep timber tones naturally present.
- **Editing:** Subtle warmth grade. Never oversaturated. Shadows retain detail.

---

### Image Category Direction

| Category | Direction |
|---|---|
| Suburb / Location | Tree-lined streets, dappled golden light, park paths, creek trails |
| Exterior Renders | Dusk or overcast sky. Greenery prominent. Never bare concrete. |
| Interior Renders | Flooded with natural light. No harsh shadows. Warm neutrals throughout. |
| Material Details | Extreme close-up. Stone grain, timber ring, bronze handle, batten shadow. |
| Lifestyle | Coffee on stone bench. Morning light through linen. Cycling trail. Never staged. |
| Food & Dining | Editorial plating. Moody. Candlelight or soft window light. |
| Education | Architectural façades. Avoid generic school photography. |
| Parks & Nature | Sunlight filtering through canopy. Seasonal colour. |

---

### Image Sizing Rules

```
Full bleed       →  Extend 3mm beyond all trim edges
Half-page        →  Exactly 50% of page width — no odd fractions
Two-thirds       →  66% of page width — the strongest split
Image captions   →  Always placed outside the image, never overlaid
White space      →  At least 8mm of breathing room around any image on a white page
```

---

### Caption Format

```
Font       →  Visby CF Light
Size       →  7.5pt
Case       →  ALL CAPS
Tracking   →  +120
Colour     →  #B9AFA3 Warm Stone
Position   →  Below image, left-aligned to image edge

Example:
ARTIST IMPRESSION — EXTERIOR FAÇADE
SCOTT GROVE STREETSCAPE — GLEN IRIS
```

## 09 — DESIGN PRINCIPLES

### The Ten Rules

```
01  Restraint is a strategy, not a limitation.
    Every element earns its place or it does not appear.

02  White space is a design element.
    Generous margins and breathing room communicate premium.

03  One typeface commands. One typeface supports.
    Clash Display leads. Visby CF follows. They do not compete.

04  The palette is a material, not a decoration.
    Every colour references something you can touch.

05  Rules are silent architecture.
    Thin lines guide the eye without demanding attention.

06  Images lead. Text explains.
    Never place competing text and image at equal visual weight.

07  Consistency builds trust.
    Caption format, rule weight, label style — never deviate.

08  Dark pages are a luxury, not a risk.
    Use them intentionally. They make light pages more luminous.

09  Never use more than three weights on one page.
    Hierarchy requires contrast, not complexity.

10  The brochure should feel like the home.
    Warm. Considered. Unhurried. Beautifully resolved.
```

## 10 — FILE & PRODUCTION NOTES

### Software
```
Primary layout    →  Adobe InDesign
Image editing     →  Adobe Photoshop  (colour grading, retouching)
Vector assets     →  Adobe Illustrator  (icons, map, logo)
Font formats      →  OTF preferred  (Clash Display, Visby CF)
```

### Colour Mode
```
Document mode   →  CMYK for print  /  RGB for digital PDF
Colour profile  →  ISO Coated v2 300% (ECI)  for offset print
Black text      →  100K only. Never rich black for type.
Rich black      →  C40 M30 Y30 K100  for large dark backgrounds only
```

### Image Specifications
```
Resolution      →  300 DPI minimum at final print size
Format          →  TIFF or PSD for print  /  JPG 85%+ for digital
Colour space    →  CMYK for print  /  sRGB for digital
Renders         →  Supplied as PNG with alpha where required
```

### Export Settings
```
Print PDF       →  PDF/X-4  300 DPI  Bleed 3mm  Marks off
Digital PDF     →  PDF 1.7  150 DPI  Hyperlinks active  Spreads off
Spreads         →  Supply as single pages to printer
                →  Export as spreads for digital viewing version
```

### Fonts — Licensing Note
```
Clash Display   →  Free for personal and commercial use (The Pangram Pangram Foundry)
Visby CF        →  Commercial license required (Connary Fagen Type Design)
Embed fonts     →  Always embed in final PDF export
```
